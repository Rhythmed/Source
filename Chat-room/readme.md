### C/S 架构 在线聊天室

Python基于Tkinter实现的C/S架构在线聊天室，包括聊天室以及私聊功能，支持文件共享，实现了用户注册、登录功能，并对距上次登录时间超出一定期限的用户进行提醒并冻结账号
